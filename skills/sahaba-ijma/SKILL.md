# Sahaba Ijma Search Skill

البحث في إجماع الصحابة وأقوالهم باستخدام API موقع الدرر السنية.

## الاستخدام

```bash
bash scripts/sahaba_search.sh "نص البحث"
```

مثال:
```bash
bash scripts/sahaba_search.sh "إجماع الصحابة"
bash scripts/sahaba_search.sh "الصحابة بايعوا أبا بكر"
```

## الخرج

يعرض كل نتيجة وتشمل:
- **نص الأثر/الإجماع**
- **الراوي** (الصحابي أو التابعي)
- **المحدث**
- **المصدر**
- **الصفحة/الرقم**
- **خلاصة حكم المحدث**

## المبدأ

> إجماع الصحابة حجة شرعية — لكننا نوثقه بالمصدر والسند.
> ما ثبت بسند متصل — أوردناه.
> ما لم يصح — نذكر درجة الصحة ولا نرفعه لإجماع.

## التوثيق

- API Endpoint: `https://dorar.net/dorar_api.json?skey=QUERY&callback=?`
- يبحث في شروح الأحاديث والآثار المروية عن الصحابة
