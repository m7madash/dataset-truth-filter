# Dorar Hadith Search Skill

البحث عن الأحاديث النبوية والتحقق من صحتها باستخدام API موقع الدرر السنية (dorar.net).

## الاستخدام

### البحث عن حديث
```bash
bash scripts/dorar_search.sh "نص البحث"
```

مثال:
```bash
bash scripts/dorar_search.sh "خيركم من تعلم القرآن"
```

### البحث بالرقم (من رابط dorar.net)
```bash
bash scripts/dorar_search.sh "114964"
```

## الخرج

يعرض كل نتيجة وتشمل:
- **نص الحديث**
- **الراوي**
- **المحدث** (البخاري، مسلم، الترمذي...)
- **المصدر**
- **الصفحة/الرقم**
- **خلاصة حكم المحدث** (صحيح، حسن، ضعيف...)

## المبدأ

> لا نُقدم حديثاً بغير علم. نتحقق من المصدر ودرجة الصحة قبل النشر.
> ما صح بسند متصل إلى صحابي — أوردناه.
> ما لم يصح — نقول "لم يثبت" ولا نفتري على الله.

## التوثيق

- API Endpoint: `https://dorar.net/dorar_api.json?skey=QUERY&callback=?`
- الكلمات المفتاحية تُكتب بالعربية
- يدعم البحث بأي كلمة من نص الحديث
